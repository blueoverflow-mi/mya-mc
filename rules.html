法律&規則

    Server機制

  • 此 server 為白名單制,如有小號請在米亞 Discord 群留下小號之 username。
  • Server IP 是可以公開的,容許非 DDoS 之攻擊,
    在發現安全漏洞時在米亞 DC 群中提醒。
  • Server 只容許玩家使用一同編寫的作弊程式,並需將程式碼在群組內公開 (open source)
    相關程式不得為其他玩家帶來困擾。
    除非被批准,否則嚴禁使用任何作弊程式,
    包括但不限於 autofish、x-ray、AFK 自動行走、autoblockplacement,killaura、reach hack、fly hack 等。
    一經發現永久封禁。

    • 遊玩規則

  • 禁止任何導致 server lag 機的建築或行為。
  • 為保障創作自由和避免遊戲變得沒趣,server不設領地系統。
    請自律不為其他玩家帶來困擾。
    不得盜取他人物件、生物或破壞他人建築物。
    如使用別人農田和動物農場後,請復原。
    如任何生物(如 Creeper) 破壞了地形或他人建築物,請自行復原或通知 admin 處理。
  • 如出現糾紛,請聯絡 admin 處理。
  • 會視乎案件嚴重性處理,由寫閱讀報告到封禁不等。請在犯罪前三思。

    • 其他

  • 米亞女王直播遊玩或拍攝時，
    請同時遵守米亞的規則
    不要干擾/作出任何騷擾行為。
  • 可以在米亞DC Minecraft 群中提出獲取地圖之請求,admin 會盡快處理。

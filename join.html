入境教學

  • 取得居留權

    此伺服器採取白名單制
    加入米亞官方DC群組，於 #一起minecraft吧！ 提交申請

    申請格式:
    dc name: xxx
    minecraft id: yyy (id 參見下面section Step 2)

  • 取得遊戲

    此伺服器版本為 java edition 1.20.6
    Step 1:
    下載 Minecraft Launcher

    Step 2:
    打開Minecraft Launcher，左邊選擇MINECRAFT: Java Edition
    綠圈為Minecraft id



    Step 3:
    按安裝檔案(Installations) -> 新安裝檔案(New Installation)



    Step 4:
    名稱(NAME) 輸入1.20.6，版本(VERSION) 選擇 release 1.20.6，然後按 儲存(Create)



    Step 5:
    選擇剛才的檔案右邊的 開始遊戲(Play) 進入遊戲


  • 加入伺服器

    Step 6:選擇多人遊戲



    選取 新增伺服器(Add server):
